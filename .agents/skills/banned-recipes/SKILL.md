# Banned Recipes Skill

This skill contains the list of recipes that are strictly banned from the Crosty family's meal plans. These recipes must never be selected by the meal selection engine, and must never be imported from the web.

## Banned Recipes List
- **Vegan Cream of Coriander (Cilantro) Soup Recipe**: Banned because the family does not like coriander/cilantro as a primary soup base.
- **Vegan Cream of Coriander (Cilantro) Soup**: Banned name variant.
- **Coriander Soup**: Banned name variant.
- **Cilantro Soup**: Banned name variant.
- **Cream of Cilantro Soup**: Banned name variant.
- **Creamy Coriander Soup**: Banned name variant.
